# face_actions_tracker
# Cyberpunk Focus & Fatigue Tracker Pro

Цей проєкт — це інтелектуальна система комп'ютерного зору, яка в реальному часі відстежує рівень уважності та втоми користувача за допомогою веб-камери. Написана на Python з використанням бібліотеки **MediaPipe Face Mesh**.

Програма автоматично калібрується під обличчя користувача, трекає напрямок погляду, повороти голови, кліпання та навіть виявляє мікросон і позіхання.



##  Основні фічі (Features)

* **Автокалібрування (Auto-Calibration):** При старті програма аналізує обличчя протягом 50 кадрів, щоб встановити ідеальні пороги для конкретної людини та освітлення.
* **Трекінг уважності (3D Gaze & Head Tracking):** Відстежує рух зіниць (X/Y) та повороти голови (Pitch/Yaw). Якщо користувач відвертається від екрана — лунає попереджувальний сигнал.
* **Детектор мікросну (Microsleep Alarm):** Якщо очі залишаються закритими більше 1.5 секунди, вмикається різка сирена для пробудження.
* **Детектор позіхання (Yawn Detection):** Аналізує MAR (Mouth Aspect Ratio) для фіксації ознак втоми.
* **Розумний лічильник кліпань:** Фільтрує мікро-шуми камери та ігнорує звичайне примружування, фіксуючи лише повноцінні кліпання (EAR - Eye Aspect Ratio).
* **Focus Score (Рейтинг уважності):** По завершенню сесії генерує детальну статистику з відсотком фокусу на екрані.

##  Вимоги до системи (Requirements)

* Веб-камера
* ОС Windows (для роботи звукової бібліотеки `winsound`)
* **Python 3.12** (Протестовано та оптимізовано для цієї версії)
